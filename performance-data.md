# Performance Data & Tracking

## Progress Tracking

### Key Metrics to Monitor
- **Bike:** FTP (Functional Threshold Power)
- **Run:** Pace at lactate threshold, threshold heart rate
- **Swim:** CSS (Critical Swim Speed)
- **Overall:** HRV, Resting Heart Rate, Sleep Quality
- **Body Composition:** Weight, body fat percentage

### Testing Schedule
- **Weeks 4, 12, 20, 28:** FTP test, run threshold test, CSS test
- **Weekly:** HRV and resting heart rate monitoring
- **Monthly:** Body composition assessment
