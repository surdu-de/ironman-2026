## Training Philosophy

This plan follows a structured periodization approach with **3 weeks of progressive build followed by 1 week of recovery**, allowing for optimal adaptation and preventing overtraining.

### Priority Focus Areas

1. **Bike Performance** - Primary focus for time improvement
2. **Run & VO2 Max** - Secondary focus for endurance and speed
3. **Swim Technique** - Continuous refinement

## Training Zones

### Bike
- **Z1:** Active Recovery (< 55% FTP)
- **Z2:** Endurance (56-75% FTP)
- **Z3:** Tempo (76-90% FTP)
- **Z4:** Threshold (91-105% FTP)
- **Z5:** VO2 Max (106-120% FTP)

### Run
- **Z1:** Recovery (< 70% HRmax)
- **Z2:** Easy/Aerobic (70-80% HRmax)
- **Z3:** Tempo (81-90% HRmax)
- **Z4:** Threshold (91-95% HRmax)
- **Z5:** VO2 Max (95-100% HRmax)

### Swim
- **T1:** Recovery/Easy
- **T2:** Aerobic/Endurance
- **T3:** Tempo
- **T4:** Threshold
- **T5:** VO2 Max/Sprint

## Nutrition Strategy

### Training Nutrition
- Carbohydrate intake: 5-7g/kg body weight on moderate days, 7-10g/kg on high-volume days
- Protein: 1.6-2.0g/kg body weight daily
- Hydration: Monitor urine color and weight changes

### Race Day Nutrition Plan
- **Pre-Race:** High carb, low fiber breakfast 3-4 hours before start
- **Swim:** No nutrition needed
- **Bike:** 60-90g carbs/hour
- **Run:** 30-60g carbs/hour
- **Hydration:** 500-750ml/hour with electrolytes

## Progress Tracking

### Key Metrics to Monitor
- **Bike:** FTP (Functional Threshold Power)
- **Run:** Pace at lactate threshold
- **Swim:** CSS (Critical Swim Speed)
- **Overall:** HRV, Resting Heart Rate, Sleep Quality
- **Body Composition:** Weight, body fat percentage

### Testing Schedule
- **Weeks 4, 12, 20, 28:** FTP test, run threshold test, CSS test
- **Weekly:** HRV and resting heart rate monitoring
- **Monthly:** Body composition assessment
