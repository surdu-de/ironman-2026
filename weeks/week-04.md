###### ğŸŠ CSS TEST (20:00-21:00)
**Purpose:** Establish swim training zones
- 400m warm-up
- **400m time trial** - record time in seconds (T400)
- 10 minutes rest (easy swimming or treading water)
- **200m time trial** - record time in seconds (T200)
- 200m cool-down

**CSS Calculation:**
```
CSS (seconds per 100m) = (T400 - T200) / 2
CSS (min:sec per 100m) = Convert to minutes:seconds format
```

**Example:** T400 = 600s (10:00), T200 = 280s (4:40)
CSS = (600 - 280) / 2 = 160s = 2:40/100m
